Summary: Latitude vs. X


The purpose of this project is to present the trends of how weather changes as you get closer to the equator. In order to accomplish this, the data is pulled from the OpenWeatherMap API to assemble a data set on over 500 random cities from all over the world.

Matplotlib was used to plot varius weather triats versus the latitude. These traits included temperature, cloudiness, wind speed, and humidity.
